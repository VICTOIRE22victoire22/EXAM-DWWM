INSTALLATION DES DEPENDANCES



Installer les dépendances frontend (Vite + React + React Router)

npm create vite@latest
cd mon-projet-vite
npm install
npm install react-router

Installer les dépendances backend

npm install express mongoose cors nodemon



Lancement du projet en ligne :


Exécution des commandes suivantes dans le terminal pour le frontend :

cd react
ls
cd recipes
ls
cd front
npm run dev

Celà donne l'URL suivante : http://localhost:5173/


On divise le terminal en 2 parties.


Exécution des commandes suivantes dans le terminal pour le backend :                                        

npm i cors
cd ..
ls
cd back
npx nodemon

Celà donne l' URL suivante : http://localhost:3000
