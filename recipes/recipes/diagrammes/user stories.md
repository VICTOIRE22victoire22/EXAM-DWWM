# User stories

Les US permettent. de décrire les fonctionnalités d'une application dans un scénario. Ils sont également un excellent moyen de définir les objectifs de notre application.

Ils sont structurés de la manière suivante :

- Rôle
- Action
- Objectif

---

## Recipes

| Rôle                         | Action (Je veux…)                       | Objectif (Afin de…)                              |
| ---------------------------- | --------------------------------------- | ------------------------------------------------ |
| En tant qu’internaute        | Consulter la liste des recettes         | Découvrir des idées de plats                     |
| En tant qu’internaute        | Rechercher une recette                  | Trouver rapidement un plat spécifique            |
| En tant qu’internaute        | Filtrer les recettes par catégorie      | Affiner ma recherche selon mes goûts             |
| En tant qu’internaute        | Voir le détail d’une recette            | Obtenir les ingrédients et étapes de préparation |
